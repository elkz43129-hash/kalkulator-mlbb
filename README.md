# 🎮 Kalkulator Win Rate MLBB 🚀

> "Capek ngitung manual sisa win biar WR kelihatan seger? I got you covered." 😉✨

Sebuah web kalkulator simpel, bersih, dan estetik buat ngitung berapa kali Anda harus **Win Beruntun** tanpa lose demi ngejar target Win Rate idaman di Mobile Legends. 

## ✨ Features
- **Anti Ribet:** Tinggal masukin total match, WR sekarang, sama target WR. Boom! Keluar hasilnya.
- **MLBB Vibes Layout:** Desain tema gelap (*dark mode*) yang ramah di mata pas lagi nge-game malam-malam.
- **Mobile Friendly:** Responsif banget, dibuka lewat HP ga bakal berantakan.
- **Built on Mobile:** 100% dideploy dan dikoding langsung lewat HP. *Peak efficiency!* 🤫🧋

## 🛠️ Tech Stack
- HTML5
- CSS3 (Custom Neon Glow Style)
- Vanilla JavaScript (Real-time Math formula)
